# NAH - No Awkward Hesitation

A high-end, exclusive landing page for a private English tutoring brand. Built with React, Vite, Tailwind CSS, and Framer Motion.

## Features

- **Dark Luxury Design**: Premium dark mode aesthetic with glassmorphism effects
- **Smooth Animations**: Powered by Framer Motion for an exclusive feel
- **Responsive Layout**: Fully responsive design that works on all devices
- **Interactive Components**: 
  - Animated hero section with cycling text
  - Bento grid feature cards with hover effects
  - Interactive fluency check recorder UI
  - Premium pricing tiers with gradient borders

## Tech Stack

- **React 19** - UI framework
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **Lucide React** - Icon library

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

```
src/
├── components/
│   ├── Navbar.jsx       # Floating navigation with glass effect
│   ├── Hero.jsx         # Hero section with animated text
│   ├── Features.jsx     # Bento grid pain points section
│   ├── FluencyCheck.jsx # Interactive recorder UI
│   ├── Pricing.jsx      # Premium pricing tiers
│   └── Footer.jsx       # Minimalist footer
├── App.jsx              # Main app component
├── index.css            # Global styles and Tailwind imports

```
